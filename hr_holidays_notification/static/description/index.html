HR Holidays Notification

Notify employees by mail on Leave Requests

Features:

☑ Send a notification to employee's manager to Approve or Decline the Request via e-mail when the Leave created.
☑ Send a notification to employee when the leave Approved.
☑ Send a notification to employee when the leave Rejected.
